# 时间管理模块指引

本文档描述时间管理模块的设计方案，用于统一管理系统时间相关功能。

---

## 一、需求背景

### 1.1 当前实现

| 位置 | 当前实现 | 问题 |
|------|----------|------|
| 首页时间显示 | 前端从 `api.uuni.cn` 获取 | 硬编码URL，无法配置 |
| 水印时间显示 | Jinja2 `now` 变量 | 使用服务器本地时间，未统一管理 |
| 系统设置 | 无时间相关配置 | 需要新增 |

### 1.2 需要解决的问题

1. 时间服务器URL硬编码，无法灵活配置
2. 多个位置使用时间的方式不统一
3. 缺乏时区管理功能

---

## 二、系统设置

### 2.1 新增设置项

在 `app/services/core/settings_service.py` 的 `DEFAULT_SETTINGS` 中添加：

```python
# ========== 时间管理 ==========
'enable_time_sync': 'true',        # 是否启用时间同步
'time_server_url': 'https://api.uuni.cn/api/time',  # 时间服务器URL
'time_zone': 'Asia/Shanghai',     # 时区设置
```

### 2.2 设置项说明

| 设置键 | 标签 | 默认值 | 说明 |
|--------|------|--------|------|
| `enable_time_sync` | 启用时间同步 | `true` | 开关，关闭则使用本地时间 |
| `time_server_url` | 时间服务器URL | `https://api.uuni.cn/api/time` | 可配置 |
| `time_zone` | 时区 | `Asia/Shanghai` | 时区设置 |

### 2.3 常用时间服务器推荐

| 服务器 | URL | 说明 |
|--------|-----|------|
| 优艾网 | `https://api.uuni.cn/api/time` | 国内服务，稳定 |
| 百度 API | `http://api.baidu.com/time/get` | 国内大厂 |
| WorldTimeAPI | `http://worldtimeapi.org/api/timezone/Asia/Shanghai` | 国际开源 |

## 三、时间服务模块

### 3.1 服务文件

创建 `app/services/core/time_service.py`：

```python
# 文件路径：app/services/core/time_service.py
# 功能说明：时间服务 - 统一管理系统时间获取

from datetime import datetime, timezone, timedelta
from typing import Optional
import requests
from app.services import SettingsService


class TimeService:
    """时间服务 - 统一管理系统时间获取"""
    
    DEFAULT_SERVERS = [
        'https://api.uuni.cn/api/time',
        'http://api.baidu.com/time/get',
        'http://worldtimeapi.org/api/timezone/Asia/Shanghai',
    ]
    
    @staticmethod
    def is_sync_enabled() -> bool:
        """是否启用时间同步"""
        setting = SettingsService.get_setting('enable_time_sync')
        return setting is None or setting == 'true'
    
    @staticmethod
    def get_time_server_url() -> str:
        """获取配置的时间服务器URL"""
        url = SettingsService.get_setting('time_server_url')
        if url:
            return url
        # 如果未配置，遍历尝试默认服务器
        for server in TimeService.DEFAULT_SERVERS:
            if TimeService._try_server(server):
                return server
        return TimeService.DEFAULT_SERVERS[0]
    
    @staticmethod
    def _try_server(url: str) -> bool:
        """测试时间服务器是否可用"""
        try:
            response = requests.get(url, timeout=2)
            return response.status_code == 200
        except Exception:
            return False
    
    @staticmethod
    def fetch_beijing_time() -> Optional[datetime]:
        """从时间服务器获取北京时间"""
        if not TimeService.is_sync_enabled():
            return None
        
        server_url = TimeService.get_time_server_url()
        
        try:
            response = requests.get(server_url, timeout=3)
            if response.status_code == 200:
                data = response.json()
                
                # 解析不同服务器返回格式
                # api.uuni.cn: {"date": "2026-03-14 12:00:00"}
                # baidu: {"t": "1699999999"}
                # worldtimeapi: {"datetime": "2026-03-14T12:00:00+08:00"}
                
                date_str = data.get('date') or data.get('datetime', '').split('+')[0]
                if date_str:
                    return datetime.strptime(date_str, '%Y-%m-%d %H:%M:%S')
                    
        except Exception as e:
            pass
        
        return None
    
    @staticmethod
    def get_current_time() -> str:
        """获取当前时间（统一入口）
        
        Returns:
            str: 格式化的当前时间字符串 "YYYY-MM-DD HH:MM:SS"
        """
        # 尝试从服务器获取时间
        server_time = TimeService.fetch_beijing_time()
        if server_time:
            return server_time.strftime('%Y-%m-%d %H:%M:%S')
        
        # 降级使用本地时间
        return datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    
    @staticmethod
    def get_timezone() -> str:
        """获取配置的时区"""
        return SettingsService.get_setting('time_zone') or 'Asia/Shanghai'
```

## 四、API 路由

### 4.1 路由文件

创建 `app/modules/core/time/__init__.py`：

```python
# 文件路径：app/modules/core/time/__init__.py
from app.modules.core.time.routes import time_bp

__all__ = ['time_bp']
```

创建 `app/modules/core/time/routes.py`：

```python
# 文件路径：app/modules/core/time/routes.py
# 功能说明：时间相关 API 路由

from flask import Blueprint, jsonify
from app.services.core.time_service import TimeService


time_bp = Blueprint('time', __name__, url_prefix='/api/time')


@time_bp.route('/current')
def get_current_time():
    """获取当前时间 API
    
    返回当前时间，支持时间服务器同步
    
    Response:
        {
            "time": "2026-03-14 12:00:00",
            "timezone": "Asia/Shanghai",
            "synced": true/false
        }
    """
    synced = TimeService.is_sync_enabled() and TimeService.fetch_beijing_time() is not None
    
    return jsonify({
        'time': TimeService.get_current_time(),
        'timezone': TimeService.get_timezone(),
        'synced': synced,
    })


@time_bp.route('/test')
def test_time_server():
    """测试时间服务器连接
    
    Response:
        {
            "success": true/false,
            "server": "https://api.uuni.cn/api/time",
            "time": "2026-03-14 12:00:00" / null
        }
    """
    server_url = TimeService.get_time_server_url()
    server_time = TimeService.fetch_beijing_time()
    
    return jsonify({
        'success': server_time is not None,
        'server': server_url,
        'time': server_time.strftime('%Y-%m-%d %H:%M:%S') if server_time else None,
    })
```

### 4.2 注册蓝图

在 `app/routes/__init__.py` 中添加：

```python
from app.modules.core.time import time_bp

# 注册蓝图
app.register_blueprint(time_bp)
```

## 五、前端改造

### 5.1 JavaScript 改造

修改 `app/static/js/common.js` 中的 `updateBeijingTime` 函数：

```javascript
// 获取并显示北京时间
window.FFE.updateBeijingTime = function() {
    const timeElement = document.getElementById('current-beijing-time');
    if (!timeElement) return;

    // 从后端API获取时间
    fetch('/api/time/current')
        .then(response => response.json())
        .then(data => {
            if (data.time) {
                timeElement.textContent = data.time;
            }
        })
        .catch(error => {
            console.error('获取时间失败:', error);
            // 降级使用本地时间
            const now = new Date();
            const weekdays = ['星期日', '星期一', '星期二', '星期三', '星期四', '星期五', '星期六'];
            const year = now.getFullYear();
            const month = String(now.getMonth() + 1).padStart(2, '0');
            const day = String(now.getDate()).padStart(2, '0');
            const weekday = weekdays[now.getDay()];
            const hours = String(now.getHours()).padStart(2, '0');
            const minutes = String(now.getMinutes()).padStart(2, '0');
            const seconds = String(now.getSeconds()).padStart(2, '0');
            timeElement.textContent = `${year}-${month}-${day} ${weekday} ${hours}:${minutes}:${seconds}`;
        });
};
```

## 六、水印改造

### 6.1 模板修改

修改 `app/templates/includes/watermark.html`：

由于水印是服务端渲染，需要在上下文处理器中添加时间：

```python
# app/__init__.py 上下文处理器中添加
@app.context_processor
def inject_utilities():
    from app.services.core.time_service import TimeService
    return dict(
        # ... existing
        current_time=TimeService.get_current_time(),
    )
```

模板中使用：

```jinja2
{% if 'datetime' in wm_content_list %}
    {% set watermark_parts = watermark_parts + [current_time] %}
{% endif %}
```
