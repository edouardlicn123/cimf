## 2. 系统性能优化

### 2.1 问题分析

经过代码审查，发现以下性能问题：

| 优先级 | 问题 | 位置 | 影响 |
|--------|------|------|------|
| **P0** | 上下文处理器频繁查询数据库 | `app/__init__.py` | 每个请求执行 4+ 次 DB 查询 |
| **P1** | SettingsService 每次查询全表 | `settings_service.py` | 高并发下数据库压力大 |
| **P1** | TimeService 每次请求外部服务器 | `time_service.py` | 高并发时严重阻塞 |
| **P2** | 重复导入 datetime | `app/__init__.py` | 代码冗余 |
| **P2** | User 模型缺少 updated_at 字段 | `user.py` | 更新用户时报错 |

### 2.2 上下文处理器优化

**问题位置：** `app/__init__.py` 第 149-188 行

**问题描述：**
- 每个请求都会调用 4 次数据库查询
- `SettingsService.get_setting()` × 2 次
- `SettingsService.get_all_settings()` × 1 次
- `NodeTypeService.get_all()` × 1 次

**优化方案：** 添加缓存机制

```python
# 在 app/__init__.py 或单独的工具模块中

from functools import lru_cache
import time

class ContextCache:
    """上下文缓存管理器"""
    
    _cache = {
        'settings': {'data': None, 'timestamp': 0},
        'node_types': {'data': None, 'timestamp': 0},
        'role_labels': {'data': None, 'timestamp': 0},
    }
    CACHE_TTL = 60  # 缓存 60 秒
    
    @classmethod
    def get_settings(cls, force=False):
        """获取设置，带缓存"""
        now = time.time()
        if force or not cls._cache['settings']['data'] or \
           (now - cls._cache['settings']['timestamp']) > cls.CACHE_TTL:
            cls._cache['settings']['data'] = SettingsService.get_all_settings()
            cls._cache['settings']['timestamp'] = now
        return cls._cache['settings']['data']
    
    @classmethod
    def invalidate_settings(cls):
        """清除设置缓存"""
        cls._cache['settings']['data'] = None
```

**问题描述：**

```python
    """系统设置服务层"""
    
    # 类级别的缓存
    _cache = {
        'all_settings': {'data': None, 'timestamp': 0},
    }
    CACHE_TTL = 60  # 缓存 60 秒
    
    @staticmethod
    def get_all_settings(as_dict: bool = True) -> Dict[str, Any]:
        now = time.time()
        cache = SettingsService._cache['all_settings']
        
        # 检查缓存是否有效

            (now - cache['timestamp']) < SettingsService.CACHE_TTL):
            return cache['data'] if as_dict else SystemSetting.query.all()
        
        # ... 原有的查询逻辑 ...
        
        # 更新缓存
        result = {}
        # ... 填充结果 ...
        cache['data'] = result
        cache['timestamp'] = now
        
        return result if as_dict else settings
    
    @staticmethod
    def clear_cache():
        """清除缓存"""
        SettingsService._cache['all_settings']['data'] = None
```

## 3. 时钟同步服务

### 3.1 背景与目标

在 08_时间管理模块指引.md 中实现了 TimeService，但存在以下问题：

| 问题 | 描述 | 影响 |
|------|------|------|
| 频繁 HTTP 请求 | 每个页面请求都可能调用外部时间服务器 | 高并发时严重阻塞 |
| 无失败重试 | 外部请求失败时无补偿机制 | 时间获取不稳定 |
| 无缓存管理 | 每次都发起请求，无本地缓存 | 性能浪费 |

**优化目标：**

1. **后台定时同步** - 每 15 分钟与远程时间服务器同步一次
2. **失败重试机制** - 同步失败时自动重试 5 次
3. **统一时间查询** - 其他服务通过本服务获取时间，不再直接请求远程服务器
4. **实时时间计算** - 基于同步时间点动态计算当前时间，保证准确性

### 3.3 TimeSyncService 实现

```python
# 文件路径：app/services/core/time_sync_service.py
# 功能说明：时钟同步服务 - 后台定时与远程服务器同步时间

import time
import logging
import threading
from datetime import datetime, timedelta
from typing import Optional
from urllib.request import urlopen
import json

from app.services.core.settings_service import SettingsService

logger = logging.getLogger(__name__)


class TimeSyncService:
    """时钟同步服务"""
    
    DEFAULT_SYNC_INTERVAL = 15 * 60      # 15分钟
    DEFAULT_MAX_RETRIES = 5
    DEFAULT_RETRY_DELAY = 2
    DEFAULT_SERVER_URL = 'https://api.uuni.cn/api/time'
    
    BACKUP_SERVERS = [
        'https://api.uuni.cn/api/time',
        'http://api.baidu.com/time/get',
        'http://worldtimeapi.org/api/timezone/Asia/Shanghai',
    ]
    
    def __init__(self):
        self._synced_time: Optional[datetime] = None
        self._last_sync_timestamp: Optional[float] = None
        self._sync_status: str = 'never'
        self._sync_thread: Optional[threading.Thread] = None
        self._running: bool = False
    
    def is_enabled(self) -> bool:
        setting = SettingsService.get_setting('time_sync_enabled')
        return setting is None or setting is True or setting == 'true'
    
    def get_sync_interval(self) -> int:
        interval = SettingsService.get_setting('time_sync_interval')
        if interval and isinstance(interval, int):
            return interval * 60
        return self.DEFAULT_SYNC_INTERVAL
    
    def get_max_retries(self) -> int:
        retries = SettingsService.get_setting('time_sync_max_retries')
        if retries and isinstance(retries, int):
            return retries
        return self.DEFAULT_MAX_RETRIES
    
    def get_server_url(self) -> str:
        url = SettingsService.get_setting('time_server_url')
        return url or self.DEFAULT_SERVER_URL
    
    def _fetch_time_from_server(self, url: str) -> Optional[datetime]:
        try:
            with urlopen(url, timeout=3) as response:
                if response.status == 200:
                    data = json.loads(response.read().decode('utf-8'))
                    date_str = data.get('date') or data.get('datetime', '').split('+')[0]
                    if date_str:
                        return datetime.strptime(date_str, '%Y-%m-%d %H:%M:%S')
        except Exception as e:
            logger.warning(f"从 {url} 获取时间失败: {e}")
        return None
    
    def _try_sync_with_servers(self) -> bool:
        servers = [self.get_server_url()] + self.BACKUP_SERVERS
        servers = list(dict.fromkeys(servers))
        max_retries = self.get_max_retries()
        
        for attempt in range(max_retries):
            for server_url in servers:
                server_time = self._fetch_time_from_server(server_url)
                if server_time:
                    self._synced_time = server_time
                    self._last_sync_timestamp = time.time()
                    self._sync_status = 'success'
                    return True
            
            if attempt < max_retries - 1:
                time.sleep(self.DEFAULT_RETRY_DELAY)
        
        self._sync_status = 'failed'
        return False
    
    def sync_time(self) -> bool:
        if not self.is_enabled():
            return False
        return self._try_sync_with_servers()
    
    def get_current_time(self) -> datetime:
        if (self._synced_time is not None and 
            self._sync_status == 'success' and 
            self._last_sync_timestamp is not None):
            elapsed = time.time() - self._last_sync_timestamp
            return self._synced_time + timedelta(seconds=elapsed)
        return datetime.now()
    
    def get_current_time_str(self, fmt: str = '%Y-%m-%d %H:%M:%S') -> str:
        return self.get_current_time().strftime(fmt)
    
    def _sync_loop(self):
        while self._running:
            try:
                self.sync_time()
            except Exception as e:
                logger.error(f"时间同步循环异常: {e}")
            time.sleep(self.get_sync_interval())
    
    def start(self):
        if self._running:
            return
        self.sync_time()
        self._running = True
        self._sync_thread = threading.Thread(target=self._sync_loop, daemon=True)
        self._sync_thread.start()
    
    def stop(self):
        self._running = False
        if self._sync_thread:
            self._sync_thread.join(timeout=5)
    
    def get_status(self) -> dict:
        return {
            'status': self._sync_status,
            'synced_time': self._synced_time.strftime('%Y-%m-%d %H:%M:%S') if self._synced_time else None,
            'last_sync_timestamp': self._last_sync_timestamp,
            'is_running': self._running,
            'enabled': self.is_enabled(),
        }


_time_sync_service: Optional[TimeSyncService] = None

def get_time_sync_service() -> TimeSyncService:
    global _time_sync_service
    if _time_sync_service is None:
        _time_sync_service = TimeSyncService()
    return _time_sync_service
```

### 3.4 TimeService 修改

```python
# 文件路径：app/services/core/time_service.py
# 修改说明：查询入口改为调用 TimeSyncService

from datetime import datetime
from app.services.core.settings_service import SettingsService
from app.services.core.time_sync_service import get_time_sync_service


class TimeService:
    """时间服务 - 统一管理系统时间获取"""
    
    @staticmethod
    def is_sync_enabled() -> bool:
        return get_time_sync_service().is_enabled()
    
    @staticmethod
    def get_time_server_url() -> str:
        return get_time_sync_service().get_server_url()
    
    @staticmethod
    def get_current_time() -> str:
        return get_time_sync_service().get_current_time_str('%Y-%m-%d %H:%M:%S')
    
    @staticmethod
    def get_current_datetime() -> datetime:
        return get_time_sync_service().get_current_time()
    
    @staticmethod
    def get_timezone() -> str:
        return SettingsService.get_setting('time_zone') or 'Asia/Shanghai'
    
    @staticmethod
    def get_sync_status() -> dict:
        return get_time_sync_service().get_status()
```
